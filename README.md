# AntonyLeons Tap

## How do I install these formulae?

`brew install antonyleons/tap/<formula>`

Or `brew tap antonyleons/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "antonyleons/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
